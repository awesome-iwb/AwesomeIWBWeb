# Redesign Backend UIs with Xiaomi HyperOS Design Language

## Goal
Redesign all backend UIs (admin, dev, operations) with Xiaomi HyperOS design language: fixed sidebar layout, scrollable content, rounded cards, glassmorphism, proper overflow handling.

## Key Issues to Fix
1. AdminShell: uses h-screen with overflow-hidden but main content needs scroll container
2. Sidebar: needs glass effect and proper width
3. Card: needs glass variant as default
4. ListDetailLayout: needs scrollable lists
5. design-tokens.css: needs HyperOS color palette

## Implementation Plan

### 1. Redesign design-tokens.css
- Add HyperOS warm neutral colors
- Add glassmorphism CSS variables
- Add rounded corner tokens (xl/full)
- Update brand colors for both admin (warm) and dev (cool)

### 2. Update AdminShell.vue
- Fix layout: fixed sidebar + scrollable main area
- Remove h-screen constraint on main content
- Add proper flexbox with min-h-0 for scroll container
- Ensure router-view has scrollable parent

### 3. Update Sidebar.vue
- Add glass effect (backdrop-blur, semi-transparent bg)
- Fix sticky positioning
- Ensure proper width and overflow handling

### 4. Update Card.vue
- Make glass variant the default
- Add more rounded corners (xl/full)
- Add glass shadow effects

### 5. Update ListDetailLayout.vue
- Ensure list area is scrollable independently
- Fix mobile layout transitions
- Add glass effect to list containers

### 6. Update all views
- DashboardView.vue: glass cards, rounded sections
- ProjectsView.vue: glass list items, rounded editor
- AnalyticsView.vue: glass chart containers
- DevDashboardView.vue: glass stat cards
- ReviewView.vue: glass panels
- All other admin/dev views

### 7. Fix scrolling issues
- Main content area should scroll, not entire page
- ListDetailLayout list should scroll independently
- BottomNav should be fixed, not cause page scroll

- Shadows: soft, colored brand shadows
